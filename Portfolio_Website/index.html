(June 2024 - August 2024)

Position: Junior Developer Intern

My primary project was to explore the capabilities of a RTOS made using different libaries and languages. This was done using a Nucleo H7 as a testing device, and a Nucleo L4 as the device being tested. The testing device was made in C with FreeRTOS, and the L4 was made in bare metal C with just HAL, C with FreeRTOS, and RUST with RTIC. My job was to compare the jitter, bandwidth, and latency of the three different programs on the L4. Load was applied using CRC calulations, GPIO responses, I2C, and UART, and was all interrupt-driven.

In this internship I became familiar with interrupt-driven programming, C, Rust, FreeRTOS, RTIC, Embassy, I2C, UART, SPI, ADC, and hardware timers.

I built a program that could tell the difference between a picture of a tomato and an apple with a neural network. This was achieved in Python and using the PyTorch libary.
